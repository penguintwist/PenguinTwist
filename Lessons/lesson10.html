Learn how to display strings, integers, floats, and booleans together!

📚 Introduction

Now that you know about strings, integers, floats, and booleans, it's time to learn how to display them all together! In real programs, you'll often need to print different types of data in the same message.

Python makes it easy to mix and match different data types when printing - let's learn how!

🔬 Theory

The Four Main Data Types:

📝

String

"Hello World"

Text in quotes
🔢

Integer

42

Whole numbers
📊

Float

3.14

Decimal numbers
✅

Boolean

True

True or False

Method 1: Using commas in print()

# Different data types name = "Alice" age = 16 height = 1.65 is_student = True # Print with commas (adds spaces automatically) print("Name:", name) print("Age:", age, "years old") print("Height:", height, "meters") print("Student:", is_student)

Method 2: Using f-strings (recommended)

# Same variables as above name = "Alice" age = 16 height = 1.65 is_student = True # F-strings for clean formatting print(f"Hi, I'm {name}") print(f"I am {age} years old") print(f"My height is {height}m") print(f"Am I a student? {is_student}")

Method 3: Converting with str()

# Convert everything to strings for concatenation score = 95 percentage = 87.5 passed = True # Convert numbers and booleans to strings print("Score: " + str(score)) print("Percentage: " + str(percentage) + "%") print("Passed: " + str(passed))
💡 Tip: F-strings are usually the best choice! They're clean, readable, and automatically convert different data types to strings for you.
