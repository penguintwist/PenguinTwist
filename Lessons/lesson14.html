Lesson 14: Concatenating Numbers

Learn how to combine numbers with text to create meaningful messages!

← Back to All Lessons

📚 Introduction

When you're working with numbers in your programs, you often want to display them alongside text. This means combining numbers with strings to create readable messages and reports.

Think about displaying a score ("You scored 85 points!") or showing a calculation result ("The total is £24.50"). This is where concatenating numbers with strings becomes essential!

🔬 Theory

The Problem: Numbers vs Strings

Numbers and strings are different data types. You can't directly concatenate them with the + operator:

# This will cause an error! age = 16 message = "I am " + age + " years old" # ERROR!

Solution 1: Convert Numbers to Strings

# Convert number to string with str() age = 16 score = 85.5 message1 = "I am " + str(age) + " years old" message2 = "You scored " + str(score) + " points" print(message1) print(message2)

Solution 2: Use F-Strings (Recommended)

# F-strings automatically convert numbers age = 16 score = 85.5 price = 24.99 message1 = f"I am {age} years old" message2 = f"You scored {score} points" message3 = f"The price is £{price}" print(message1) print(message2) print(message3)

Solution 3: Use Commas in Print

# Print handles different data types automatically age = 16 height = 1.75 name = "Alex" print("Name:", name, "Age:", age, "Height:", height, "metres") print("Total score:", 95, "out of", 100)

✅ F-strings (Best)

  • Clean and readable
  • Automatic type conversion
  • Easy formatting options
  • Modern Python standard

⚠️ str() Method

  • Works but gets messy
  • Manual conversion needed
  • More typing required
  • Error-prone with many numbers
⚠️ Remember: You cannot use + to directly combine strings and numbers. You must convert the number to a string first, or use f-strings which do it automatically.
💡 Tip: F-strings are usually the best choice for combining numbers and text. They're cleaner, easier to read, and handle the conversion automatically!
