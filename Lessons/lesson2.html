📚 Introduction

Variables are like labeled boxes where you can store information. Just like you might have a box labeled "Toys" or "Books", in programming we create labeled containers for our data.

Think of a variable as a name tag that you stick on a piece of information so you can find it later!

🔬 Theory

Creating a variable in Python is simple. You just choose a name and assign a value to it:

name = "Alice" age = 15 print("My name is", name) print("I am", age, "years old")

Variable rules:

  • Variable names can contain letters, numbers, and underscores
  • They must start with a letter or underscore (not a number)
  • No spaces allowed - use underscores instead (like my_name)
  • Python is case-sensitive: Name and name are different!
💡 Tip: Use descriptive names for your variables. student_name is much clearer than just n!
