üìö Introduction

In the real world, we use different types of numbers for different purposes. When counting things (like apples, books, or people), we use whole numbers - no fractions or decimals needed!

In Python, these whole numbers are called "integers" or "int" for short. They're one of the most important data types you'll use in programming.

üî¨ Theory

Integers in Python are whole numbers that can be positive, negative, or zero:

# Positive integers age = 16 score = 100 # Negative integers temperature = -5 debt = -50 # Zero is also an integer balance = 0 print(f"Age: {age}") print(f"Temperature: {temperature}") print(f"Balance: {balance}")
üìñ What makes an integer:
  • No decimal point (42, not 42.0)
  • Can be positive, negative, or zero
  • No fractions (use 5, not 5/2)
  • Can be very large (Python handles big numbers automatically)

Checking if something is an integer:

number = 42 print(type(number)) # Shows: # Converting strings to integers age_text = "25" age_number = int(age_text) print(f"Age as number: {age_number}")
üí° Tip: Use the type() function to check what data type a variable is. Use int() to convert strings to integers!
