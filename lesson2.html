Why Do Programs Need Memory?

In Lesson 1, you learned to make the computer display messages. But what if you want the computer to remember something - like your name - and use it multiple times?

Variables are like labeled boxes in the computer's memory. You can store information in these boxes and use it whenever you need it.

Understanding Variables

Think of a variable as a labeled storage box:

name
"Alex"
age
16
school
"Greenwood High"

To create a variable in Python, you use this pattern:

name = "Alex"

This tells Python: "Create a box labeled 'name' and put 'Alex' inside it."

Step-by-Step Process: 1. Python sees name = and creates a new storage box labeled "name"
2. Python sees "Alex" and puts this text into the box
3. Now whenever you use name, Python looks in that box and gets "Alex"
Key Rule: Variable names have NO quotes around them. The values you store IN variables need quotes if they're text.

Using Variables with Print

Once you've stored something in a variable, you can use it in print statements:

name = "Alex" age = 16 print(name) print(age)
Output: Alex 16

Notice the difference:

Code What it does Output
print("name") Prints the word "name" name
print(name) Prints what's stored in the name variable Alex
Common Mistake: Putting quotes around variable names. print("name") prints "name", but print(name) prints what's stored in the variable.

Changing Variable Values

One powerful feature of variables is that you can change what's stored in them:

pet = "cat" print(pet) pet = "dog" print(pet)
Output: cat dog

When you assign a new value to an existing variable, the old value is completely replaced:

pet (first)
"cat"
→
pet (after change)
"dog"
Important: The computer doesn't remember the old value. When you change a variable, the previous value is gone forever. The variable only stores the most recent value you gave it.

You can create as many variables as you need:

name = "Sarah" age = 15 school = "Oak Valley School" subject = "Computer Science" print(name) print(age) print(school) print(subject)
Output: Sarah 15 Oak Valley School Computer Science
