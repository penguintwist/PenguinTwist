üìö Introduction

Imagine you're getting dressed in the morning. You look outside - if it's raining, you take an umbrella. If it's sunny, you wear sunglasses. Programs need to make decisions too!

That's where "if statements" come in. They let our programs choose what to do based on different conditions.

üî¨ Theory

An if statement checks a condition and runs code only if it's true:

weather = input("Is it sunny or rainy? ") if weather == "sunny": print("Great! Let's go to the park!") else: print("Let's stay inside and read a book!")

Key parts of if statements:

  • if - starts the condition check
  • weather == "sunny": - the condition to check (note the colon!)
  • == means "is equal to" (we use two equals signs)
  • Indented code runs only if the condition is true
  • else: - what to do if the condition is false

Let's see another example:

answer = input("What is 5 + 3? ") if answer == "8": print("Correct! Well done!") else: print("Not quite right. The answer is 8.")
üí° Tip: Python uses indentation (spaces) to group code together. Always indent the code that goes with your if statement by pressing the space bar 4 times!
