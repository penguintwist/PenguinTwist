📚 Introduction

Sometimes we need to work with numbers that aren't whole - like prices (£3.99), measurements (1.75 meters), or temperatures (36.7°C). These numbers with decimal points are called "floating-point numbers" or "floats" in Python.

Floats let us be much more precise in our calculations than integers alone!

🔬 Theory

Float numbers in Python represent decimal values and can be positive, negative, or zero:

# Positive floats height = 1.75 price = 9.99 pi = 3.14159 # Negative floats temperature = -5.2 debt = -12.50 # Zero as a float balance = 0.0 print(f"Height: {height} meters") print(f"Price: £{price}") print(f"Temperature: {temperature}°C")
📖 What makes a float:
  • Has a decimal point (3.14, not 3)
  • Can represent fractions and precise values
  • Perfect for measurements, money, percentages
  • Can be very small (0.001) or very large (999.999)

Working with floats:

# Check the data type weight = 65.5 print(type(weight)) # Shows: # Convert strings to floats height_text = "1.80" height_number = float(height_text) print(f"Height: {height_number}m") # Convert integers to floats age = 16 age_float = float(age) print(f"Age as float: {age_float}")

Rounding floats:

price = 19.99567 rounded_price = round(price, 2) # Round to 2 decimal places print(f"Original: {price}") print(f"Rounded: {rounded_price}")
💡 Tip: Use float() to convert strings or integers to floats. Use round(number, places) to round to a specific number of decimal places!
