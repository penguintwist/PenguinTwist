📚 Introduction

In life, many questions have simple yes/no answers: "Are you hungry?" "Is it raining?" "Did you finish your homework?" In programming, we use Boolean values to represent these True or False answers.

Booleans are named after George Boole, a mathematician who created Boolean algebra. They're essential for making decisions in programs!

🔬 Theory

Boolean values in Python are simple - there are only two possibilities:

# Boolean values is_student = True is_raining = False finished_homework = True weekend = False print(f"Am I a student? {is_student}") print(f"Is it raining? {is_raining}") print(f"Finished homework? {finished_homework}")
📖 Boolean Rules:
  • True and False must be capitalized exactly
  • No quotes needed - they're special keywords, not strings
  • Used for making decisions in if statements
  • Result of comparisons (5 > 3 gives True)

Boolean Comparisons:

# Comparisons return Boolean values age = 16 print(age > 18) # False print(age >= 16) # True print(age == 16) # True print(age != 20) # True # String comparisons name = "Alice" print(name == "Alice") # True print(name == "Bob") # False
✅ TRUE Examples
5 > 3
"cat" == "cat"
16 >= 16
not False
❌ FALSE Examples
10 < 5
"dog" == "cat"
15 > 20
not True

Converting to Boolean:

# Check data type is_sunny = True print(type(is_sunny)) # Shows: # Convert to boolean number = 5 text = "hello" empty = "" print(bool(number)) # True (non-zero numbers) print(bool(text)) # True (non-empty strings) print(bool(empty)) # False (empty strings) print(bool(0)) # False (zero)
💡 Tip: Use bool() to convert values to True/False. Most values are True, but 0, empty strings "", and None are False!
